# Inventory Management System

A simple console-based Inventory Management System built with Python.

## Features

- Add Product
- View Products
- Search Product
- Delete Product
- Update Product Quantity
- Sell Product
- Product Statistics
- Save data to JSON
- Load data automatically on startup

## Product Data

Each product stores:

- Name
- Price
- Quantity

Example:

{
    "MOUSE": {
        "price_value": 50,
        "quantity_value": 10
    }
}
